# Classic Solutions

* Download the project folder
* Navigate to the file directory in terminal

## Setting up

* Install all node elements

    ```
    npm install
    ```
    
## Build

To build the project 

* Build Server

    ```
    npm build
    ```

## Usage

There's a couple of methods to getting the server running

* **Server does not run using eduroam WIFI**
* Run Server. The application will automatically reload if you change any of the source files.

    ```
    npm run dev
    ```

    

* Run standard 

    ```
  node server
    ```
 ## Testing
